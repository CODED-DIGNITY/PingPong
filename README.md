# 🏓 PingPong

A simple Pong-style game built with [raylib](https://www.raylib.com/).  
Play against a friend and relive the classic arcade vibes.

Included cmakelists if using Clion.

## 🕹️ Controls
- **W / S** → Move left paddle  
- **Up / Down** → Move right paddle
